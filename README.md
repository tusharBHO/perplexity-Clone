# 🤖 AI Platform (Perplexity × ChatGPT Hybrid)

An AI-powered platform inspired by **Perplexity** and **ChatGPT**, built with Next.js, TailwindCSS, and LLM APIs (OpenAI, Gemini, etc).  
🚀 A showcase project for full-stack development skills.

---

## ✨ Features
- 🔍 Ask anything with a clean query UI (like Perplexity)
- 🤝 Multiple LLM APIs (GPT, Gemini, etc.)
- 📚 Library for saving Q&As
- 🔑 Authentication (Clerk)
- 📊 Clean UI powered by **TailwindCSS + shadcn/ui**
- ⚡ Built on **Next.js App Router**

---

## 🛠 Tech Stack
- **Framework**: Next.js 15
- **Styling**: TailwindCSS, shadcn/ui
- **Auth**: Clerk
- **LLMs**: OpenAI, Gemini
- **Hosting**: Vercel (recommended)
